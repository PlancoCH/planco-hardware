## Purpose

This repository contains all CAD and PCB design files for the Planco hardware device, including 3D models, schematics, PCB layouts, and bill of materials. It serves as the central hub for hardware documentation and design iterations.

## Prerequisites

Before working with the hardware files, ensure you have the following installed:

* Autodesk Fusion 360 (for CAD models)
* EasyEDA (for PCB design and schematics)
* Access to Planco's internal infrastructure

## Installation & Setup

Step-by-step instructions to set up your hardware design environment.

```bash
# Clone the repository
git clone git@github.com:PlancoCH/planco-hardware.git

```

## Usage

How to access and work with the hardware files.

* Open CAD files in **Autodesk Fusion 360**
* Open PCB and schematic files in **EasyEDA**
* Refer to the project's folder structure for organizing components, assemblies, and documentation
